# 4-bit Ripple Carry Adder (Verilog)

## 📌 Overview
This project implements a 4-bit Ripple Carry Adder using Verilog HDL.

Simulated using **EDA Playground + EPWave**.

---

##  Files
- ripple_carry_adder_4bit.v → Design
- tb_ripple_carry_adder.v → Testbench
- waveform.png → Simulation waveform screenshot

---

##  Design Concept
- Uses 4 full adders
- Carry propagates from LSB → MSB
- Basic combinational circuit design

---

## How to Run
1. Open EDA Playground
2. Paste design + testbench
3. Select simulator (Icarus Verilog)
4. Run simulation
5. Open EPWave for waveform

---

## Output
- Console output shows input & result
- EPWave shows waveform transitions
- Screenshot included in repo

##  Learning Outcome
- Verilog RTL design
