# Logic Gates Verification using Verilog

## Overview
Designed and verified basic logic gates (AND, OR, NOT) using Verilog HDL with a self-checking testbench.

## Key Features
- Dataflow modeling using `assign`
- Task-based testbench
- Automatic PASS/FAIL validation
- Waveform generation (VCD)

## Results
All input combinations were tested successfully and passed verification.

## Tools
Verilog HDL, Icarus Verilog / ModelSim, GTKWave
