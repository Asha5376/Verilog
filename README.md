# Verilog
MY VLSI Verification journey

This repository contains my Verilog HDL practice projects designed
and simulated using EDA Playground (Icarus Verilog + EPWave).

## Each Project Folder Includes
- Verilog RTL design code
- Testbench code
- Simulation output
- Waveform screenshot (EPWave)

All designs are verified using simulation and waveform analysis.

## Tools Used
- Verilog HDL
- EDA Playground
- Icarus Verilog Simulator
- EPWave

## Purpose
To practice digital design concepts, improve Verilog coding skills,
and understand simulation and debugging flow.
This repository is maintained for continuous learning and practice.
